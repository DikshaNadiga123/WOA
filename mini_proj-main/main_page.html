What is Whale Optimization Algorithm?

The whales are considered to be as highly intelligent animals with motion. The WOA is inspired by the unique hunting behavior of humpback whales. Usually the humpback whales prefer to hunt krills or small fishes which are close to the surface of sea. Humpback whales use a special unique hunting method called bubble net feeding method. In this method they swim around the prey and create a distinctive bubbles along a circle or 9-shaped path. The mathematical model of WOA is described in the following sections 1. Encircling prey. 2. Bubble net hunting method. 3. Search the prey.

Define the Problem:

Objective Function

  • Power Loss: The objective could be minimizing the total active and reactive power losses in the distribution system.
  • Cost: The cost associated with the installation, operation, and maintenance of DGs.
  • Voltage Stability: Ensuring that voltage levels at each bus are stable and within acceptable limits.

Variables

  • DG Location: The optimal location(s) of Distributed Generators (DGs) in the grid.
  • DG Capacity: The power capacity of each DG to install.

Constraints

  • Voltage Limits: Voltage at each node should stay within acceptable limits.
  • Power Flow: The power generated should not exceed the power demand.
  • DG Capacity: The size of each DG should not exceed its maximum capacity.

Whale Optimization Algorithm (WOA) Steps

  1. Initialization:
    • Initialize a population of whales (solutions) randomly.
    • Each whale represents a solution with a candidate location and size for the DGs.
  2. Fitness Evaluation:
    • Calculate the fitness of each whale based on the chosen objective function (e.g., power loss or cost).
  3. Exploration and Exploitation:
    • Exploration: Whales search for global optima by randomly adjusting their positions, simulating the search for prey.
    • Exploitation: Whales refine their search by updating their position toward the best solution found so far (like hunting the prey).
  4. Update Position:
    • The position of each whale is updated using the WOA formula, which mimics the behavior of humpback whales during hunting.
    • The movement of a whale is determined by its distance from the prey (best solution), and it will either encircle or approach the prey.
  5. Termination:
    • The algorithm stops after a predefined number of iterations or once convergence criteria are met.

  1. Application to DG Optimization:

    • Objective Function Example: Minimize the power loss in the grid due to improper DG placement. This can be expressed as:
      Objective Function = ∑lines Power Loss (P) = f(DG Location, DG Capacity)
  2. Optimization Process:
    • WOA evaluates multiple locations for Distributed Generators (DGs).
    • Both positions (locations) and capacities are adjusted to minimize the power loss or cost.

  3. Output:

    • The optimized locations and capacities of DGs that achieve the objective (e.g., reduced power loss or cost).
  4. Summary:
    • WOA helps in finding optimal locations and sizes for DGs in a distribution network.
    • It balances the trade-off between reducing losses, improving voltage profiles, and minimizing costs.



Whale Optimization Algorithm (WOA) for DG Optimization

The Whale Optimization Algorithm is applied to find the best locations and capacities for Distributed Generators (DGs) in a power distribution system. Here’s what each parameter means:

  • Number of Buses: Number of connection points (buses) in the system.
  • Resistances: The resistive properties of the transmission lines between buses.
  • Population Size: Number of candidate solutions (whales) evaluated during each iteration.
  • Max Iterations: Maximum number of algorithm cycles (iterations) to refine solutions.
  • Min DG Size: The smallest allowed DG capacity (in kW) for installation.
  • Max DG Size: The largest allowed DG capacity (in kW) for installation.
  • Best Solution: The optimal placement and sizing of DGs found by the algorithm.
  • Minimum Loss: The goal of minimizing power losses in the system by placing DGs optimally.


INPUT AND OUTPUT OF WOA DG

1. What the 15 Numbers Represent

  • Each of the 15 values corresponds to the size of a Distributed Generator (DG) placed at a particular bus in the power system.
  • The values represent the amount of power (in kW or MW) generated by the DG at each bus to minimize the total system power loss.
  • Example: If the result is 60, 120, 0, 0, 3000, ..., this means:
    • A 60 kW DG is placed at bus 1.
    • A 120 kW DG is placed at bus 2.
    • No DG is placed at buses 3 and 4 (indicated by 0).
    • A 3000 kW DG is placed at bus 5, and so on.

2. What These Numbers Mean

  • Positive Values: Each positive value represents the size (or power capacity) of the DG installed at that specific bus. These values range from the minimum DG size to the maximum DG size based on the optimization constraints.
  • Example: A value of 3000 indicates a 3000 kW DG is installed at the bus (maximum allowable size).
  • Zero Values: A value of 0 means that no DG has been placed at that bus. It indicates that placing a DG there is not optimal for minimizing system losses.

3. How They Help in Minimizing Loss

  • The algorithm determines the optimal placement and size of DGs for each bus to minimize the total system loss.
  • The output of the 15 numbers shows how much power each bus will generate if a DG is placed there, which contributes to minimizing power losses.
  • These numbers are part of the best solution, where DGs are strategically placed and sized to reduce the total system power loss.

4. How to Interpret the Output

  • If the output contains mostly zeros and a few non-zero values, it means DGs are placed only at specific buses where they have the greatest impact on reducing losses.
  • The algorithm avoids placing DGs at buses where their presence does not significantly reduce system losses.
  • The values should adhere to the constraints of minimum and maximum DG sizes defined in the optimization problem.

Example of Output

  • Let's assume the output for a system with 15 buses is:
    • 0, 1000, 0, 0, 500, 0, 0, 0, 3000, 0, 2000, 0, 0, 0, 1500
  • This means:
    • Bus 1: No DG (0 kW).
    • Bus 2: A 1000 kW DG is installed.
    • Bus 3: No DG (0 kW).
    • Bus 4: No DG (0 kW).
    • Bus 5: A 500 kW DG is installed.
    • Bus 6: No DG (0 kW).
    • Bus 7: No DG (0 kW).
    • Bus 8: No DG (0 kW).
    • Bus 9: A 3000 kW DG is installed (maximum size).
    • Bus 10: No DG (0 kW).
    • Bus 11: A 2000 kW DG is installed.
    • Bus 12: No DG (0 kW).
    • Bus 13: No DG (0 kW).
    • Bus 14: No DG (0 kW).
    • Bus 15: A 1500 kW DG is installed.

Conclusion

  • The minimum loss output is a configuration that provides the optimal placement and sizing of DGs to minimize power loss in the distribution system.
  • Non-zero values represent the DG sizes at the respective buses, while zero values indicate no DGs placed at those buses.
  • This configuration ensures that the system has minimal power loss while satisfying constraints such as the minimum and maximum DG size limits.
